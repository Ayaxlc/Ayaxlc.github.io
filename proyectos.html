Fresh Market

Fresh Market es un proyecto de diseño web que simula un pequeño supermercado online. El objetivo principal del proyecto es practicar la estructura de páginas HTML, la maquetación con CSS y la navegación entre distintas secciones y productos, aplicando conceptos de diseño de interfaces y usabilidad.

StepUpStore

StepUpStore es una aplicación web de comercio electrónico desarrollada con PHP y MySQL. Permite la gestión de usuarios (clientes y administrador), productos, carrito de compra y pedidos. El proyecto simula una tienda online real trabajando con sesiones, roles y conexión a base de datos.
